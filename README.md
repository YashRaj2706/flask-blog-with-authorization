# Flask Blog App (Login + Dashboard + CRUD)

## 📌 Project Description

This is a simple Flask-based web application that allows users to:

* Register and login
* Create posts
* View posts on dashboard
* Delete their own posts

## 🚀 Features

* User authentication (login/register)
* Session management
* Create and delete posts
* SQLite database using Flask-SQLAlchemy

## 🛠️ Tech Stack

* Python
* Flask
* SQLite
* HTML/CSS (Bootstrap)

## 📂 Project Structure

createpost/

├── app.py

│
├── templates/
   ├── login.html

   ├── dashboard.html
   └── createpost.html


## ▶️ How to Run

1. Clone the repository
2. Install dependencies:

   ```
   pip install -r requirements.txt
   ```
3. Run the app:

   ```
   python app.py
   ```
4. Open browser:

   ```
   http://127.0.0.1:5000
   ```
